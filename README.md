## Descripción de componentes

### Arduino UNO
Microcontrolador principal encargado de la lectura de sensores, procesamiento de datos y control de actuadores del sistema automatizado.

### Pantalla LCD 16x2 I2C
Permite visualizar en tiempo real las variables ambientales, estados del sistema y parámetros configurables del ascensor industrial.

### Sensor DHT22
Sensor digital utilizado para medir temperatura y humedad relativa del entorno de operación.

### Sensor LDR
Sensor resistivo dependiente de la luz empleado para detectar el nivel de iluminación ambiental y automatizar el sistema lumínico.

### Receptor infrarrojo (IR)
Dispositivo utilizado para recibir órdenes remotas desde un mando infrarrojo, permitiendo modificar parámetros del sistema sin interacción física directa.

### LEDs de señalización
Indicadores visuales utilizados para representar estados de funcionamiento, alarmas o activación de sistemas auxiliares.

### Buzzer
Actuador acústico utilizado para generar alertas sonoras ante determinadas condiciones del sistema.
